Collega il tuo sito WordPress alla piattaforma Webmaster Monitor per monitoraggio completo.

== Description ==

Webmaster Monitor è un plugin che permette di collegare il tuo sito WordPress alla piattaforma di monitoraggio Webmaster Monitor.

= Funzionalità =

* **Informazioni Server**: PHP version, MySQL/MariaDB, web server, spazio disco
* **Informazioni WordPress**: versione, plugin, temi, utenti
* **Aggiornamenti**: notifica plugin e temi da aggiornare
* **Site Health**: integrazione con WordPress Site Health
* **API Sicura**: autenticazione tramite API key univoca

= Requisiti =

* WordPress 5.0 o superiore
* PHP 7.4 o superiore
* Account su Webmaster Monitor

== Installation ==

1. Scarica il plugin dalla piattaforma Webmaster Monitor
2. Vai su Plugin > Aggiungi nuovo > Carica plugin
3. Seleziona il file ZIP e clicca "Installa ora"
4. Attiva il plugin
5. Vai su Impostazioni > Webmaster Monitor
6. Copia l'API Key e inseriscila nella piattaforma

== Frequently Asked Questions ==

= È sicuro? =

Sì, il plugin utilizza un'API key univoca per ogni sito. Solo chi possiede l'API key può accedere ai dati.

= Quali dati vengono raccolti? =

Il plugin raccoglie informazioni tecniche sul server e su WordPress: versioni software, plugin installati, temi, spazio disco. Non raccoglie dati personali degli utenti del sito.

= Posso rigenerare l'API key? =

Sì, dalla pagina Impostazioni > Webmaster Monitor puoi rigenerare l'API key in qualsiasi momento.
